= Popup Generation =

Popup Generation - the simple and functional WordPress plugin for creating pop-ups. With the popup plugin, you can easily create modal windows, pop-ups and lightboxes for your own needs as you wish and then insert anywhere place on the site. 

* Create any kind of modal windows with a powerful editor
* Style them how you want
* Place them anywhere
* Insert any content
* Analytics for each popup
* Tooltips to create popup

= Form Generation =

Form Generation - the simple and functional WordPress plugin for creating custom forms. With the form plugin, you can easily create any forms for your own needs as you wish and then insert anywhere place on the site. Use Analytics for each form for reach marketing targets.

* Powerful Custom Style Editor
* Enable powerful email settings
* Collect emails & other data
* Create any amount of forms
* RECAPTCHA
* Analytics for each form
* Tooltips to create form

= Button Generation =

Create unique custom buttons with icons. Easily insert button in any place what you want. Create standard and floating buttons with statistics.

* Powerful Button Builder
* Style them how you want

== Installation ==

* Installation option 1: Find and install this plugin in the `Plugins` -> `Add new` section of your `wp-admin`
* Installation option 2: Download the zip file, then upload the plugin via the wp-admin in the `Plugins` -> `Add new` section. Or unzip the archive and upload the folder to the plugins directory `/wp-content/plugins/` via ftp
* Press `Activate` when you have installed the plugin via dashboard or press `Activate` in the in the `Plugins` list 
* Go to `LeadGeneration` section that will appear in your main menu on the left
* Choose under `LeadGenerationP` the tool you want to work with
* Click `Add new` to create your first item
* Setup your item
* Click Save
